# Enterprise Management Systems (Banking & Hospital) 🏦🏥

**Domain:** Core Java Development

## 🚀 Project Overview
Comprehensive desktop software applications developed for Banking and Hospital administration to streamline operations and secure data.

## 💻 Tech Stack
* **Language:** Core Java
* **GUI Framework:** Java Swing & AWT
* **Database:** MySQL
* **Database Connectivity:** JDBC API

## ⚙️ Key Implementations
* Developed comprehensive desktop software applications for Banking and Hospital administration utilizing Java Swing and AWT interfaces.
* Engineered secure relational database schemas in MySQL, utilizing the JDBC API for seamless data persistence, retrieval, and real-time record synchronization.
* Implemented core software functionalities including role-based access control, PIN-based authentication, and automated transaction reporting.
* Applied rigorous Object-Oriented Programming (OOP) principles to create modular, scalable, and reusable code blocks.
